🚫 Social Media Content Blocker

Your browser extension is ready! Here's how to install and use it.

🔧 Installation Instructions

1
Open Chrome and go to chrome://extensions/
2
Enable "Developer mode" toggle in the top-right corner
3
Click "Load unpacked" and select the browser-extension folder
4
Pin the extension by clicking the puzzle piece icon in Chrome's toolbar
5
Visit X (Twitter) or LinkedIn to test the blocker!
✅ Core Features Implemented
  • 🎥 Video/Reel Blocking: Automatically hides video content from feeds
  • 🔍 Keyword Filtering: Block posts containing specific words or phrases
  • 👤 User Blocking: Manually block specific users by username
  • 📊 Real-time Statistics: Track how much content has been blocked
  • ⚙️ Easy Management: Simple popup and advanced settings page
  • 🔄 Dynamic Monitoring: Works with infinite scroll and dynamic content
🚧 Advanced Features (Next Phase)
  • 👑 Influencer Blocking: Block users with >50K followers (requires follower scraping)
  • 🤖 AI Content Analysis: Smart detection of sponsored/promotional content
  • 📱 More Platforms: Instagram, TikTok, Facebook support
  • ⏰ Scheduled Blocking: Time-based rules (e.g., block entertainment during work hours)
🎉 Ready to Test! The extension is fully functional for keyword and video blocking on X (Twitter) and LinkedIn. The core architecture is in place to add more advanced features.

🧪 Testing Instructions

After installing the extension:

  1. Configure Settings: Click the extension icon and enable blocking features
  2. Add Keywords: Try adding "sponsored", "crypto", or "NFT" as blocked keywords
  3. Visit X/Twitter: Go to twitter.com or x.com and scroll through your feed
  4. Visit LinkedIn: Go to linkedin.com and check your feed
  5. Check Statistics: See the counter in the popup and blocked content overlays
  6. Advanced Settings: Click "Advanced Settings" for comprehensive management
⚠️ Note: Follower count detection is currently simulated with random numbers for testing purposes. A production version would implement proper follower count scraping or API integration.
